# Autodidactic_CPPProject_Note
Autodidactic note of different cpp projects in modern computation, network, cpp grammar, etc.
这个项目整理了有关C++学习的几百个问题，涵盖现代cpp开发的各个方面。目的是方便需要学习CPP的初学开发人员。有少量问题前后重复，并未去除，原因是觉得重复回答也能巩固。<br>
如果能以学习为目的，逐个将所有的问题解答出来，相信能使CPP学习框架更加完善。如果为了应付近期的考试，时间十分紧迫，也可以私信我要所有问题的答案，每一道题的答案我按照自己的理解都做了解答。
